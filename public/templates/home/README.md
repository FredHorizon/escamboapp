## Dependências do projeto

* ImageMagick (https://www.imagemagick.org)

```
sudo apt-get update
sudo apt-get install imagemagick
```

* Graphviz (https://www.graphviz.org) - Apenas para Ambiente de Desenvolvimento

```
sudo apt-get update
sudo apt-get install graphviz
```
